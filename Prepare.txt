#For Ubuntu/Debian-based systems (using apt)
#The most straightforward method uses the deadsnakes PPA, which provides recent Python versions without interfering with the system's default Python installation. 

1. Update package lists and install PPA management tools:

sudo apt update
sudo apt install software-properties-common

2. Add the deadsnakes PPA:

sudo add-apt-repository ppa:deadsnakes/ppa
sudo apt update

3. Install Python 3.11 and the venv module:

sudo apt install python3.10 python3.10-venv
